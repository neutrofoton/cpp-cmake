# All about CMake

## General Note
1. In CMake C++ expressed as CXX
2. We can override value of variable built in CMake. For example
    ```bash
    set(CMAKE_CXX_STANDARD 17)
    set(CMAKE_CXX_STANDARD_RQUIRED ON)
    set(CMAKE_CXX_EXTENSIONS OFF)
    ```
